# Weather App

## 🔍 Overview
A simple web app that fetches live weather data from WeatherAPI based on user input.

## ✨ Features
- Search weather by city name (In progress)
- Displays temperature, weather condition
- Uses proxy server to protect API Key

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- WeatherAPI
